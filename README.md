## Apps of My Own
[Talk Spotter](https://github.com/EvanBoyar/talk-spotter), a voice-activated amateur radio spotting tool for Linux. Talk Spotter listens to radio audio streams, transcribes speech on-device, and can post spots to the DX Cluster network and POTA via voice commands over SSB/FM  
[The Hop Report](https://evanboyar.github.io/hopreport/), a dashboard of your local band conditions, calculated using realtime sources  
[CW Jitsu](https://github.com/EvanBoyar/cwjitsu), an Android Morse code practice app  
[Shaw CW](https://github.com/EvanBoyar/ShawCW), an Android app allowing the deaf and hard of hearing to use Morse code  
[Trigeo](https://github.com/EvanBoyar/trigeo), an Android app for transmitter searches (AKA fox hunts).  
[Solar Windsock](https://evanboyar.github.io/solar-windsock/), a  simple svg windsock that flaps harder when the solar wind blows more quickly
